# Input Template

Use this template when the user wants a reusable daily invocation or has provided only a topic.

```text
请为 TASHI SOUL 生成一篇小红书图文笔记。

【主题】

【内容目的】
品牌内容 / 生活哲思 / 情绪共鸣 / 产品种草 / 香气场景 / 节气内容 / 品牌故事

【产品露出】
不露出产品 / 轻微露出 / 明确介绍藏香 / 转化型产品内容

【目标人群】
默认：30-50 岁、身处高压环境的城市精英。

【情绪卡点】

【藏式哲理方向】
无常 / 放下 / 观照 / 慈悲 / 因缘 / 空性 / 止语 / 转念 / 接纳 / 不执着

【正文要求】
默认 120-200 字以内。

【图片数量】
3 张 / 4 张 / 6 张 / 9 张

【图片背景方向】
藏地自然 / 藏地人文 / 两者结合

【图片元素偏好】
自然元素：深绿叶片 / 静谧湖泊 / 远处雪山 / 高原云影 / 山谷晨雾 / 石头 / 风 / 草地 / 湖面
人文元素：转经筒 / 寺庙外墙 / 藏式建筑 / 经幡远景 / 祈福的人 / 行走背影 / 手部转经动作 / 安静寺院角落

【图片构图】
左侧暗区 / 右侧暗区 / 中央偏下暗区 / 大面积天空留白 / 湖面留白 / 山体暗区

【图片排版】
圆点分割 / 实线分割 / 虚线分割 / 竖排 / 分行

【标签偏好】
偏情绪疗愈 / 偏生活方式 / 偏职场压力 / 偏中年成长 / 偏藏地文化 / 偏品牌调性 / 偏产品转化

【额外要求】
不要 emoji；不要出现产品名；图片不要出现人物正脸；图片不要出现佛像正面。
```
