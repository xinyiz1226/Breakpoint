# Sample Post

## Input

```text
主题：下班后身体回家了，心还在公司
内容目的：品牌内容 / 情绪共鸣
产品露出：不露出产品
目标人群：30-50 岁、身处高压环境的城市精英
情绪卡点：停不下来，回家后仍然紧绷，手机一响就进入工作状态
藏式哲理方向：观照 / 放下
正文要求：120-200 字以内，更故事化，更生活化
图片数量：4 张
图片背景方向：藏地自然 / 藏地人文结合
图片元素偏好：静谧湖泊 / 远处雪山 / 寺庙外墙 / 转经筒
图片构图：左侧暗区或中央偏下暗区，用于放置文字
额外要求：不要 emoji，不要出现产品名，图片不要出现佛像正面，封面文案更克制。
```

## Output Shape

```text
【选题洞察】
这个主题写的是高压城市人群回家后的延迟放松：身体离开了工作，神经还停在责任和消息里。切入点不是劝他立刻放下，而是写一个人如何慢慢把心带回当下。

【产品露出判断】
本篇默认不露出产品，不出现藏香和品牌名。

【标题】
共鸣型：
1. 身体到家了，心还在公司
2. 中年人的累，是停不下来

1. 昨晚十一点，他在车里坐了很久
2. 下班后，我没有立刻上楼

轻哲理型：
1. 心静了，事才变小
2. 人不是解决完一切，才配休息

收藏型：
1. 回家，也需要时间

【封面文案】
1. 心静了，事才变小
2. 回家，也需要时间
3. 不必急着有答案

【正文】
昨晚十一点，他把车停进地库，却没有立刻上楼。

手机亮了几次，他都没看。只是坐着，听排风管很轻地响。

白天的会议、催促、没说完的话，还像风一样留在身体里。

后来他想起，山上的云不会因为被催促就散开。

人也是。

有时候，不是事情解决了，心才可以安静。
是心先安静下来，事情才回到原来的大小。

【图片脚本】
1. 藏地自然：夜色高原湖泊，远处雪山，左侧暗区。文字：心静了 · 事才变小。圆点分割。
2. 藏地人文：寺庙外墙与阴影中的转经筒，右侧暗区。文字：不必急着 · 有答案。实线分割。
3. 藏地自然：山谷晨雾，大片天空留白。文字：回家 · 也需要时间。分行排版。
4. 藏地人文：祈福的人远景背影，中央偏下暗区。文字：人先安静 · 风才会停。虚线分割。
```
