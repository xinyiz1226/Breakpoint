# TASHI SOUL Daily User Prompt Template v2

```text
请为 TASHI SOUL 生成一篇小红书图文笔记。

【主题】
填写主题。
例如：下班后身体回家了，心还在公司 / 周日晚焦虑 / 中年人的沉默 / 饭局后的疲惫 / 深夜失眠 / 关系里的边界感 / 不敢停下来

【内容目的】
品牌内容 / 生活哲思 / 情绪共鸣 / 产品种草 / 香气场景 / 节气内容 / 品牌故事

【产品露出】
不露出产品 / 轻微露出 / 明确介绍藏香 / 转化型产品内容

说明：
如果选择“不露出产品”，正文、标题、图片都不要主动出现藏香或 TASHI SOUL 产品。
如果选择“轻微露出”，可以把藏香作为生活场景中的一个动作，但不要介绍卖点。
如果选择“明确介绍藏香”，可以写香气、原料、使用场景，但仍需克制。
如果选择“转化型产品内容”，可以适度说明适合什么人、什么场景使用，但不能夸大功效。

【目标人群】
默认：30-50 岁、身处高压环境的城市精英。
也可以细分为：企业管理者 / 创业者 / 职场中高层 / 高压女性 / 中年男性 / 长期失眠人群 / 家庭责任承担者

【情绪卡点】
填写读者正在经历的具体卡点。
例如：停不下来 / 睡不着 / 总想控制结果 / 无法拒绝 / 内耗 / 害怕失败 / 被责任压着 / 下班后仍然紧绷

【藏式哲理方向】
无常 / 放下 / 观照 / 慈悲 / 因缘 / 空性 / 止语 / 转念 / 接纳 / 不执着
如果不确定，可以写：请自行选择最适合主题的哲理方向。

【正文要求】
默认 120-200 字以内。
也可以填写：更短 / 更故事化 / 更像《次第花开》的清明气质 / 更适合收藏 / 更少哲理 / 更生活化

【图片数量】
3 张 / 4 张 / 6 张 / 9 张

【图片背景方向】
藏地自然 / 藏地人文 / 两者结合

【图片元素偏好】
自然元素：
深绿叶片 / 静谧湖泊 / 远处雪山 / 高原云影 / 山谷晨雾 / 石头 / 风 / 草地 / 湖面

人文元素：
转经筒 / 寺庙外墙 / 藏式建筑 / 经幡远景 / 祈福的人 / 行走背影 / 手部转经动作 / 安静寺院角落

【图片构图】
左侧暗区 / 右侧暗区 / 中央偏下暗区 / 大面积天空留白 / 湖面留白 / 山体暗区
如果不确定，可以写：请根据画面自行安排文字暗区。

【图片排版】
圆点分割 / 实线分割 / 虚线分割 / 竖排 / 分行
如果不确定，可以写：请根据标题自行选择排版。

【标签偏好】
偏情绪疗愈 / 偏生活方式 / 偏职场压力 / 偏中年成长 / 偏藏地文化 / 偏品牌调性 / 偏产品转化

【额外要求】
填写其他限制或偏好。
例如：
- 不要 emoji
- 不要出现“疗愈”这个词
- 标题更克制
- 正文不要提品牌名
- 图片不要出现人物正脸
- 图片不要出现佛像正面
- 封面文案更像一句诗
```
