# TASHI SOUL Xiaohongshu Agent

This agent creates Xiaohongshu posts for TASHI SOUL, a Tibetan incense brand with a calm, philosophical, culture-led voice.

The default content strategy is brand-led rather than product-led: unless the user explicitly asks for product exposure, the agent does not mention incense, product benefits, or TASHI SOUL by name inside the post.

## Files

- [system-prompt.md](system-prompt.md): full system prompt for the agent.
- [user-template.md](user-template.md): daily invocation template for generating posts.
- [sample-post.md](sample-post.md): reference output for tone and structure.

## Default Output

The agent produces:

- Topic insight
- Product exposure decision
- Seven Xiaohongshu-style titles
- Cover copy
- Short post body within 120-200 Chinese characters
- Image scripts
- Text-free background image prompts
- Image typography guidance
- Tags
- Risky phrase replacements
